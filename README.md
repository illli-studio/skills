# Claude Code Skills

A collection of Claude Code skills for various tools and APIs.

## Available Skills

### ui2v-api

AI-powered video and poster generation via HTTP API.

| Feature | Description |
|---------|-------------|
| **Video Generation** | Text-to-animation video (MP4/WebM) |
| **Poster Generation** | Text-to-image (PNG/JPG) |
| **Quality Tiers** | 5 levels: low → cinema |
| **Custom Dimensions** | Configurable width/height |
| **Style System** | Preset and custom styles |

[View Skill →](./ui2v-api/SKILL.md)

## Installation

```bash
npx skills add illli-studio/skills
```

## Usage

Skills are automatically loaded when relevant tasks are detected. No manual activation required.

# Claude Code Skills 技能集合

各种工具和 API 的 Claude Code 技能集合。

## 可用技能

### ui2v-api

通过 HTTP API 实现 AI 驱动的视频和海报生成。

| 功能 | 说明 |
|------|------|
| **视频生成** | 文本转动画视频 (MP4/WebM) |
| **海报生成** | 文本转图像 (PNG/JPG) |
| **质量分级** | 5 档质量: low → cinema |
| **自定义尺寸** | 可配置宽度/高度 |
| **风格系统** | 预设风格和自定义风格 |

[查看技能 →](./ui2v-api/SKILL.md)

## 安装

```bash
npx skills add illli-studio/skills
```

## 使用方法

技能会在检测到相关任务时自动加载，无需手动激活。
